# <a name="keyboard-shortcuts-for-the-remote-control-viewer-in-system-center-configuration-manager"></a>System Center Configuration Manager 中遠端控制檢視器的鍵盤快速鍵

適用於：System Center Configuration Manager (最新分支)

當您使用 System Center Configuration Manager 遠端控制檢視器時，可以使用下列鍵盤快速鍵來控制所管理的用戶端電腦。  

|鍵盤快速鍵|說明|  
|-----------------------|-----------------|  
|Alt+Page Up|從左到右切換各執行中程式。|  
|Alt+Page Down|從右到左切換各執行中程式。|  
|Alt+Insert|依執行中程式的開啟順序，來循環使用執行中程式。|  
|Alt+Home|顯示 [開始]  功能表。|  
|Ctrl+Alt+End|顯示 [Windows 安全性] 對話方塊 (Ctrl+Alt+Del)。|  
|Alt+Delete|顯示 Windows 功能表。|  
|Ctrl+Alt+減號 (位於數字鍵台)|將本機電腦的使用中視窗複製至遠端電腦剪貼簿。|  
|Ctrl+Alt+加號 (位於數字鍵台)|將整個本機電腦的視窗區域複製至遠端電腦剪貼簿。|
